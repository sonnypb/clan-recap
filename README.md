# Clan Recap
Recap for the previous month of clan achievements

## Tutorial (Step-by-step)

Follow this exactly if you're using this for the first time.

--- 

### 1. Download the project

#### Option A (Git users)
```bash
git clone https://github.com/sonnypb/clan-recap.git
cd clan-recap
```

#### Option B (non-Git users)
- Click **Code → Download ZIP**
- Extract the folder somewhere on your PC
- Open the folder

--- 

### 2. Install Python (if you don’t already have it)

Check if Python is installed:

```bash
python --version
```

You should see something like:
```
Python 3.10.x
```

If not installed, download it from:
https://www.python.org/downloads/

---

### 3. Edit input file

Open `recap_input.txt` and fill it in like this:

```
Achievements:
Grandmaster Combat Achievements: Player1, Player2
First Infernal Cape: Player4

Pets:
Zulrah Pet: Player1
Pet Kree Arra: Player2

Drops:
Imbued Heart: Player1, Player2
```

---

### 4. Run the script

Open a terminal in the folder and run:

```bash
python recap_script.py
```

---

### 5. Enter month

When prompted, enter either:
- Full month: `January`
- Short form: `jan`, `feb`, etc.

---

### 6. Output

A file will be created in the output folder like:

```
recap_february_2026.txt
```

Copy this into Discord or wherever you post your recap!
